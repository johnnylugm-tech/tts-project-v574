# TTS Project v574 - 基於 edge-tts 之高品質簡報配音系統

## 一句話說清這事

使用 Microsoft Edge TTS API 將文字轉為高品質中文語音，支援簡報配音、批次處理、錯誤恢復。

## 使用範例

### 範例 1：完整合成流程

```python
import asyncio
from presentation_tts import PresentationTTS
from config_manager import TTSConfig

async def main():
    # 建立自訂配置
    config = TTSConfig(
        voice="zh-TW-HsiaoHsiaoNeural",
        rate="+0%",      # 語速調整
        volume="+0%",    # 音量調整
        chunk_size=500,   # 分段大小
        retry_count=3     # 重試次數
    )
    
    # 建立 TTS 實例
    tts = PresentationTTS(config)
    
    # 合成
    text = """
    第一段文字內容。
    第二段文字內容。
    第三段文字內容。
    """
    
    result = await tts.synthesize(text, "presentation.mp3")
    print(f"已完成: {result}")

asyncio.run(main())
```

### 範例 2：流式合成

```python
import asyncio

asyncio.run(stream_example())
```

## 配置說明

### TTSConfig 參數

| 參數 | 類型 | 預設值 | 說明 |
|------|------|--------|------|
| voice | str | "zh-TW-HsiaoHsiaoNeural" | 音色名稱 |
| rate | str | "-2%" | 語速調整（-50% ~ +100%） |
| volume | str | "+0%" | 音量調整（-50% ~ +50%） |
| chunk_size | int | 500 | 文本分段大小 |
| max_text_length | int | 10000 | 最大輸入字數 |
| retry_count | int | 3 | 重試次數 |
| timeout | int | 30 | 請求超時（秒） |

## 錯誤處理

### 錯誤類型（L1-L4）

| 等級 | 錯誤類型 | 說明 | 處理方式 |
|------|----------|------|----------|
| L1 | NetworkError | 網路連線失敗 | 重試 + 熔斷 |
| L2 | ServiceError | 服務端錯誤 | 重試 + 熔斷 |
| L3 | InputError | 用戶輸入錯誤 | 立即失敗 |
| L4 | SystemError | 系統錯誤 | 記錄 + 通知 |

### 使用範例

```python
from error_handler import InputError, NetworkError

try:
    result = await tts.synthesize(text, "output.mp3")
except InputError as e:
    print(f"輸入錯誤: {e}")
except NetworkError as e:
    print(f"網路錯誤: {e}")
```
